Sistema inteligente de optimización logística desarrollado con Python y Streamlit.

## Características

- Selección de almacenes solicitados por el usuario. 
- Optimización de rutas mediante A*.
- Estrategia del algoritmo del vecino más cercano.
- Visualización interactiva con PyVis.
- Interfaz web desarrollada en Streamlit.

## Tecnologías

- Python 3.12
- Streamlit
- NetworkX
- Pandas
- PyVis

## Ejecución local

```bash
pip install -r requirements.txt
streamlit run app.py
